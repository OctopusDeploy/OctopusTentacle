To install add new repository to /etc/apt/sources.list:

deb https://s3.amazonaws.com/octopus-apt-repo/ stretch main

And import key that is used to sign the release:

$ apt-key adv --fetch-keys https://s3.amazonaws.com/octopus-apt-repo/public.key

After that you can install tentacle as any other software package:

$ apt-get update
$ apt-get install tentacle
